===
Для отрисовки Кнопки скачивания категорий используется ф-ция wc_catalog_pdf_button($cat_ids).
===
Где $cat_ids - это массив ID категорий, которые выведутся в файл pdf
===
Результат разбивается на части по 500 товаров (чтобы исключить ошибку сервера)
===
Каждая кнопка выведет в файл pdf результат части (до 500) товаров
